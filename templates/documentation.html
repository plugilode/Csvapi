Query Parameters

Parameter Description Example
page Page number for pagination ?page=2
per_page Number of records per page ?per_page=20
sort_by Column name(s) to sort by (can be used multiple times) ?sort_by=name&sort_by=age
sort_order Sort direction for each sort_by field (asc or desc) ?sort_order=desc&sort_order=asc
search Search term across all columns ?search=john
fields Select specific fields to include in response (can be used multiple times) ?fields=name&fields=age
[column]__gt Greater than filter for numeric fields ?age__gt=25
[column]__lt Less than filter for numeric fields ?price__lt=100
[column]__gte Greater than or equal filter ?age__gte=18
[column]__lte Less than or equal filter ?price__lte=99.99
[column]__contains Contains text filter (case-insensitive) ?name__contains=john
[column]__startswith Starts with text filter ?name__startswith=j
[column]__endswith Ends with text filter ?email__endswith=.com
[column] Exact match filter ?city=London
